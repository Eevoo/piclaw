Project: gamelit-v1
Root: readwrite/stories/gamelit-v1/

Purpose: Develop an interactive game-lit narrative set in a Japanese-inspired fantasy world with steampunk influences. The story will support character creation, species selection, and an in-game interview process that forms the player character's character sheet. Episodes will be saved sequentially under text/ as 1.md, 2.md, etc.

Contents:
- world_brainstorm.md — high-level setting hooks, factions, tech levels, tone
- species_list.md — candidate playable species with notes on daily life and party roles
- gear_and_tech.md — steampunk elements, common devices, and their cultural impact
- economy_and_games.md — how the game works in-world (win conditions, progression, risks)
- char_sheet_template.md — live character sheet format to be updated during creation and play
- interview_questions.md — in-game interview to fill the character sheet and establish roleplaying cues

Notes:
- Save narrative outputs (scenes/episodes) to readwrite/stories/gamelit-v1/text/ as sequential numbered .md files per user preference.
- When the user requests a redo, use retry suffixes (e.g., 1-a.md) until a scene is accepted. Once accepted, increment the next file number.

Status: Initialized. Next steps: populate concept files with brainstormed content and create initial scene stub in text/1.md.
